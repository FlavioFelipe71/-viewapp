Bem-vindo ao Fleshviewapp

Este site é uma interface para rodar o Fleshviewapp localmente.

Instruções

Para executar o programa, siga os passos abaixo:

1. Baixe e instale o Python.
2. Clone este repositório:
   git clone https://github.com/flaviofelipe71/-viewapp.git
   cd -viewapp
3. Instale os requisitos:
   pip install -r requirements.txt
4. Execute o comando:
   flask --app Fleshviewapp executar
5. Abra no navegador:
   http://localhost:5000
